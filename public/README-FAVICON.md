# Favicon va Rasmlar Yo'riqnomasi

Bu papkaga quyidagi rasmlarni qo'ying:

## 1. favicon.ico
- **O'lcham**: 16x16, 32x32, 48x48 px (bir faylda)
- **Maqsad**: Brauzer tab'ida ko'rinadigan logotip
- **Format**: .ico
- **Qanday yasash**: 
  - https://favicon.io yoki https://realfavicongenerator.net dan foydalaning
  - Logoyingizni yuklang va ICO formatda yuklab oling

## 2. apple-touch-icon.png
- **O'lcham**: 180x180 px
- **Maqsad**: iPhone/iPad home screen icon
- **Format**: PNG (shaffof fon bilan)

## 3. og-image.jpg
- **O'lcham**: 1200x630 px (Facebook/social media)
- **Maqsad**: Facebook, WhatsApp, Telegram'da share qilganda ko'rinadi
- **Format**: JPG yoki PNG
- **Mazmun**: Logotip + kompaniya nomi + qisqa matn

## 4. logo.png
- **O'lcham**: 512x512 px yoki 1024x1024 px
- **Maqsad**: Yuqori sifatli logotip (kerak bo'lsa)
- **Format**: PNG (shaffof fon)

## Maslahat:
- Barcha rasmlar sifatli bo'lishi kerak
- favicon.ico uchun oddiy, aniq ko'rinadigan dizayn tanlang
- og-image.jpg da matnlar o'qilishi kerak (katta shrift)
